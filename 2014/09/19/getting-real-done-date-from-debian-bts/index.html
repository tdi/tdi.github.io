getting real "done date" of a bug from Debian BTS

As I wrote in my last post currently, SOAP interface, nor Ultimate Debian Database do not provide a date when a given bug was closed (done date). It is quite hard to calculate statistics on a bug tracker when you do not know when a bug was closed (!!).

Done date of bug can be found in its log. The log itself can be downloaded by SOAP method get_bug_log but the processing of it is quite complicated. The same comes to web scrapping of a BTS's web interface. Fortunatelly the web interface gives a possibility to download a log in an mbox format.

After analysing an example log, one can encounter a line looking like below. It means that for this bug number 657783 BTS received an email closing the bug. It can also be closed by 657783-close but for RFS it will always be done. 

Received: (at 657783-done) by bugs.debian.org; 29 Jan 2012 13:27:42 +0000

Below is a script that extracts the done date of a bug from its log in mbox format. It uses requests to download the mbox and caches the result in ./cache/bugs, which you need to create. 

 :::python
 #!/usr/bin/env python

 import requests
 from datetime import datetime
 import re
 import os

 class BugLog(object):

     CACHE_DIR="./cache/bugs/"

     def __init__(self, num):
         self._num = num

     @property
     def done_date(self):
         if os.path.exists("{}{}".format(BugLog.CACHE_DIR, self._num)):
             return self._get_from_cache(self._num)
         else:
             r = requests.get("https://bugs.debian.org/cgi-bin/bugreport.cgi?mbox=yes;bug={};mboxstatus=yes".format(self._num))
             reg = "Received: \(at {}-done\) by bugs\.debian\.org\; (\d+ \w+ \d\d\d\d) \d\d:\d\d:\d\d".format(self._num)
             result = re.search(reg, r.text)
             d = datetime.strptime(result.group(1), "%d %b %Y")
             with open("{}{}".format(BugLog.CACHE_DIR, self._num), "w") as f:
                 f.write("{}".format(d.date()))
             return d.date()

     def _get_from_cache(self, num):
         with open("{}{}".format(BugLog.CACHE_DIR, self._num)) as f:
             return datetime.strptime(f.readlines()[0], "%Y-%M-%d").date()


 if __name__ == "__main__":
     b = BugLog(657783)
     print(b.done_date)

PS: I hope that the script will be not needed in the near future, as Don Armstrong plans a new BTS database, a Debconf14 video is here.
